Lesson 03

Paths are addresses

Read file and folder locations without panic.

What you are learning

Absolute paths start from the top

/Users/mia/Desktop gives the full address from the root of the Mac.

Relative paths start where you stand

Desktop means “the Desktop folder from here”.

Home shortcuts save typing

~ means your home folder. ~/Desktop means “Desktop inside my home folder”.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd ~/Desktop
TYPE THISpwd
MAC REPLIES/Users/mia/Desktop
TYPE THIScd ~/Downloads

Teacher note

Useful shortcuts: ~ home, . here, .. one folder up.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
