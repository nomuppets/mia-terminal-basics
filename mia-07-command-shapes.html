Lesson 07

Commands have parts

Learn how to read a command before you run it.

What you are learning

The command is the action

In ls -la ~/Desktop, ls is the action.

Options change behaviour

-la asks for a longer list including hidden files.

The target says where

~/Desktop says where to run the action.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THISls -la ~/Desktop
NOTEls is the command. -la is the option. ~/Desktop is the target.

Teacher note

When a command looks scary, split it into action, options, and target.

Tiny mission

  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
