Lesson 04

Make folders and files

Create a safe practice folder on your Desktop.

What you are learning

Make a folder

Use mkdir. It means make directory.

Go inside it

Use cd terminal-practice after you create it.

Make an empty file

Use touch. It creates a blank file if the file does not exist.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd ~/Desktop
TYPE THISmkdir terminal-practice
TYPE THIScd terminal-practice
TYPE THISmkdir notes projects photos
TYPE THIStouch notes/today.txt
TYPE THISopen .

Teacher note

You made real things on your Mac. Finder will show the same folders and file.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
