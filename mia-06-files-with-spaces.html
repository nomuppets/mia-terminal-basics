Lesson 06

Files with spaces

Use quotes when names contain spaces.

What you are learning

Spaces split words

Terminal treats spaces as separators unless you protect them.

Use quotes first

cd "My Folder" is easier to read than backslashes.

Backslashes also work

cd My\ Folder means the space is part of the folder name.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd "Holiday Photos"
TYPE THISopen "My Notes.txt"
TYPE THIScd My\ Folder

Teacher note

Use quotes. They are boring and clear, which is exactly what you want.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
