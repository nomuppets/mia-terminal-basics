Cheat sheet

All the commands and symbols from the course in one safe place.

How to use this page

First check whether the item is a terminal command or a keyboard shortcut. A terminal command is text you type into iTerm2 and run with Return. A keyboard shortcut is a key, or group of keys, you press on the keyboard. ~ means your home folder, usually something like /Users/mia.

Terminal commands

Type these into iTerm2, then press Return.

pwd

Shows where Terminal is standing right now. Read it as “where am I?”

ls

Lists what is inside the current folder.

cd folder

Moves into a folder. Example: cd Desktop.

cd ..

Moves one folder up.

cd ~

Moves back to your home folder.

cd ~/Desktop

Moves to the Desktop folder inside your home folder.

open .

Opens the current folder in Finder. The dot means “here”.

mkdir name

Makes a new folder. Example: mkdir notes.

touch file.txt

Makes a blank file if it does not already exist.

rm file.txt

Deletes one file. Slow down before you press Return.

rmdir folder

Deletes an empty folder. It refuses if the folder still contains files.

clear

Cleans the screen. It does not delete files.

q

Quits many full-screen views, like help pages or logs.

Keyboard shortcuts

Press these on the keyboard. Do not type the words into iTerm2.

Tab

Completes folder and file names when Terminal can guess the rest.

↑

Brings back the previous command so you do not need to type it again.

Ctrl+C

Stops many commands that keep running or feel stuck.

Command (⌘) +

Makes the iTerm2 text bigger.

Symbols you learned

~

~ means your home folder. Example: cd ~.

.

. means the folder where Terminal is standing now. Example: open ..

..

.. means one folder above where Terminal is standing. Example: cd ...

"My Folder"

Quotes protect names with spaces. Example: cd "Holiday Photos".

Safe practice run

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd ~/Desktop
TYPE THISpwd
MAC REPLIES/Users/mia/Desktop
TYPE THISls
NOTEThe list you see depends on what is on your Desktop.

Still avoid these

If a random tutorial asks for sudo, rm -rf, chmod, or chown, stop and ask someone first. Those commands can change or delete important things.
