Lesson 10

Practice mission

Put the basics together in one safe exercise.

What you are learning

Create a practice area

Work on the Desktop so you can see what happens in Finder.

Move, list, and open

Use the commands from earlier lessons in one flow.

Explain it back

If you can explain each line, you are learning the terminal properly.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd ~/Desktop
TYPE THISmkdir mia-practice
TYPE THIScd mia-practice
TYPE THISmkdir notes screenshots projects
TYPE THIStouch notes/first-note.txt
TYPE THISls
MAC REPLIESnotes projects screenshots
TYPE THISopen .
TYPE THIScd ..
TYPE THISpwd

Teacher note

You used ~, cd, mkdir, touch, ls, and open . in one mission.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
