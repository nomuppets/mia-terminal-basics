Lesson 09

Get unstuck

Use a small rescue kit when the screen gets confusing.

What you are learning

Clear the screen

Use clear. This only cleans the view. It does not delete files.

Go home

~ means your home folder. cd ~ gets you back to a safe familiar place.

Exit or stop

Use q to leave many full-screen views. Use Ctrl+C to stop many running commands.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THISclear
TYPE THIScd ~
TYPE THISpwd
MAC REPLIES/Users/mia
NOTEIf something keeps running, press Ctrl+C.

Teacher note

You are allowed to reset your view. Calm beats panic.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
