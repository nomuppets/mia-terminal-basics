Lesson 02

Move between folders

Use cd, .., and ~ to move without getting lost.

What you are learning

Change folder

Use cd followed by the folder name. cd Desktop means “go into Desktop”.

Go one folder up

Use cd ... Two dots mean “the folder above this one”.

Go home

~ means your home folder. On Mia’s Mac, that is probably /Users/mia. cd ~ means “go home”.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd Desktop
TYPE THISpwd
MAC REPLIES/Users/mia/Desktop
TYPE THIScd ..
TYPE THIScd ~

Teacher note

You do not need to memorise long paths. ~ is the shortcut back to your home folder.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
