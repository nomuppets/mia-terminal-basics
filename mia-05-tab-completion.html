Lesson 05

Let the Mac finish your typing

Use Tab and command history to avoid mistakes.

What you are learning

Type less

Start a folder name, then press Tab. The Mac fills in the rest when it can.

Reuse commands

Press the up arrow to bring back the last command.

Stop a running command

Press Ctrl+C if a command seems stuck or keeps running.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIScd ~/Des
NOTEPress Tab. iTerm2 should complete it to cd ~/Desktop.
TYPE THIScd ~/Desktop
NOTEPress the up arrow to bring the command back.

Teacher note

Good terminal users do not type everything. They let completion do the boring work.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
