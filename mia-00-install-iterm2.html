Lesson 00

Install iTerm2

Put the right terminal app on your Mac before you learn commands.

What you are learning

Download from the official site

Open the official iTerm2 downloads page. Use that link, not a random mirror site.

Move it into Applications

Open your Downloads folder, unzip iTerm2 if your Mac has not already done it, then drag iTerm2 into Applications.

Open iTerm2

Open Applications, then open iTerm2. If macOS asks whether you trust it, confirm only because you downloaded it from the official site.

Make the text readable

Use Command (⌘) + a few times if the text feels small. Do not customise anything else yet.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THISpwd
MAC REPLIES/Users/mia
TYPE THISopen .

Teacher note

Type pwd to ask your Mac where Terminal is standing. Type open . to open that same place in Finder.

Tiny mission

  1. Read the line label first.
  2. Type only the lines marked TYPE THIS into iTerm2.
  3. Say what changed on your Mac.
