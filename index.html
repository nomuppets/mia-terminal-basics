What ~ means

~ means your home folder. On Mia’s Mac, that is probably /Users/mia. You will use it as a shortcut in commands like cd ~ and cd ~/Desktop.

Safety rule

If a tutorial asks for sudo or rm -rf, stop and ask someone first. This course teaches the safe basics before anything sharp.

Mia Terminal Basics is a beginner course for Mac users. Type slowly, read what the command says, and avoid dangerous commands you do not understand.
