Lesson 08

Delete carefully

Deletion in Terminal is real. Slow down here.

What you are learning

Delete one file

Use rm file.txt only when you know the exact file.

Remove an empty folder

Use rmdir empty-folder. It refuses if the folder still has things inside.

Pause on dangerous commands

Do not run sudo or rm -rf from a random tutorial. Ask someone first.

Try this

Type only the lines marked TYPE THIS. Do not type lines marked MAC REPLIES or NOTE.

TYPE THIStouch test-delete-me.txt
TYPE THISrm test-delete-me.txt
TYPE THISmkdir empty-folder
TYPE THISrmdir empty-folder

Teacher note

The terminal will not always ask “are you sure?” Treat delete commands with respect.

Tiny mission

  2. Type only the lines marked TYPE THIS into iTerm2.
